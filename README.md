# Fullstack Portfolio — Next.js 15

Стартовый шаблон портфолио фулстек‑разработчика на Next.js 15 + Tailwind + Framer Motion + next-themes.

## Локальный запуск
```bash
pnpm install
pnpm dev
# http://localhost:3000
```

## Продакшн
```bash
pnpm build
pnpm start
```

## Деплой на Vercel
Импортируйте репозиторий на Vercel → Deploy.
